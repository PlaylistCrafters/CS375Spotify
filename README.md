# CS375Spotify

A multiplayer music quiz game

# Stack

- Backend: Express
- Frontend: NextJS

# Installing locally

## Backend

- `cd backend`
- `npm install`

## Frontend

- `cd frontend`
- `npm install`

## Shortcut Scripts:

- Windows: `install-dependencies.bat`

# Running Locally

## Secrets
- `cd backend`
- create a file called `.env` with the following contents
```
CLIENT_ID="<Spotify Client ID>"
CLIENT_SECRET="<Spotify Client Secret>"
```

## Backend

- `cd backend`
- `node server.js`

## Frontend

- `cd frontend`
- `npm run dev`
- visit http://localhost:3000

## Shortcut Scripts

- `run.bat`
