# CS375Spotify

A multiplayer music quiz game

# Stack

- Backend: Express
- Frontend: NextJS

# Installing locally

## Backend

- `cd backend`
- `npm install`

## Frontend

- `cd frontend`
- `npm install`

## Shortcut Scripts:

- Windows: `install-dependencies.bat`

# Running Locally

## Secrets

- `cd backend`
- create a file called `.env` using `backend/.env_template` as a template
- adjust secrets as necessary

- `cd frontend`
- create a file called `.env.local` using `frontend/.env_template`
- adjust secrets as necessary

## Backend

- `cd backend`
- `node server.js`

## Frontend

- `cd frontend`
- `npm run dev`
- visit http://localhost:3000

## Shortcut Scripts

- `run.bat`
