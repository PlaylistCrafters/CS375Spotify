# CS375Spotify

A multiplayer music quiz game

# Stack

- Backend: Express
- Frontend: NextJS

# Installing locally

## Backend

- `cd backend`
- `npm install`

## Frontend

- `cd frontend`
- `npm install`

## Shortcut Scripts:

- Windows: `install-dependencies.bat`

# Running Locally

## Backend

- `cd backend`
- `node server.js`

## Frontend

- `cd frontend`
- `npm run dev`
- visit http://localhost:3000

## Shortcut Scripts

- `run.bat`
